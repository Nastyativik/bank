# Банковское приложение на Go

Программа представляет собой консольное приложение на языке Go для управления банковскими счетами. Приложение предоставляет интерактивное меню с различными банковскими операциями, такими как создание счета, пополнение, снятие, переводы и просмотр истории.

- Приложение запускает функцию main()
- Инициализируется банковская система с хранилищем счетов
- Запускается бесконечный цикл с отображением меню
- Пользователь выбирает действие, система выполняет соответствующую операцию
- Приложение завершается при выборе соответствующего пункта меню

## Состояния меню

- Создать счет — создание нового банковского счёта с именем владельца
- Пополнить счет — внесение средств на счёт
- Снять средства — снятие средств со счёта
- Перевести средства — перевод средств на другой счёт по ID
- Проверить баланс — отображение текущего баланса счёта
- Получить выписку — просмотр истории транзакций по счёту
- Показать все счета — отображение списка всех счетов в системе
- Выйти — завершение работы приложения

## Основные структуры данных

Transaction - Представляет одну транзакцию (пополнение, снятие, перевод).

Account - Представляет банковский счёт с уникальным ID, именем владельца, балансом и историей транзакций.

MemoryStorage - Структура, реализующая интерфейс Storage, для хранения счетов в памяти.

BankApp - Основная структура приложения, управляющая логикой работы и вводом-выводом.

## Основные интерфейсы

AccountService - Интерфейс для выполнения операций со счётом: Deposit, Withdraw, Transfer, GetBalance, GetStatement.

Storage - Интерфейс для работы с хранилищем: SaveAccount, LoadAccount, GetAllAccounts.

## Основные методы

__NewBankApp() *BankApp__

Создает и инициализирует новое банковское приложение.

__createAccount()__

Создание нового счёта с уникальным ID и именем владельца.

__deposit()__

Пополнение счёта на указанную сумму с проверкой:

- Сумма должна быть положительной
- Счёт должен существовать
  
__withdraw()__

Снятие средств со счёта с проверкой:

- Сумма должна быть положительной
- На счёте должны быть достаточные средства
- Счёт должен существовать

__transfer()__

Выполнение перевода средств с одного счёта на другой с проверкой:
 
- Сумма должна быть положительной
- Пользователь не может перевести самому себе
- На счёте отправителя должны быть достаточные средства
- Счёта отправителя и получателя должны существовать

__checkBalance()__

Отображение текущего баланса счёта.

__getStatement()__

Отображение истории транзакций по счёту.

__showAllAccounts()__

Отображение списка всех созданных в системе счетов.

__findAccount() *Account__

Поиск счёта по ID. Возвращает указатель на счёт или nil, если не найден.

__getFloatInput() (float64, error)__

Вспомогательная функция для безопасного ввода чисел с плавающей точкой.

Все операции, которые могут завершиться неудачно, возвращают ошибку с понятным описанием причины, которое отображается пользователю.
